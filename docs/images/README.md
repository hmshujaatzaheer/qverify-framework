# Documentation Images

This directory contains images for QVERIFY documentation.

## Contents

- `qverify_logo.png` - QVERIFY logo
- `architecture.png` - System architecture diagram
- `workflow.png` - Verification workflow diagram

## Creating Images

To add new images:

1. Use PNG or SVG format
2. Keep file sizes reasonable (<500KB)
3. Use descriptive filenames
4. Update this README

## Logo Specifications

- Format: PNG with transparency
- Size: 400x100 pixels (recommended)
- Colors: Use brand colors from style guide
